Placing the CSS property display: flex; on an element allows you to use other flex properties to build a responsive page.

Align Elements Using the justify-content Property

when direction is row so, justify-content:flex-start; left side justify-content:flex-end; right side justify-content:space-between; 1st to left, last to right justify-content:space-around; half space at either ends justify-content:space-evenly; equal spaces justify-content:center; box at center

CSS offers the align-items property to align flex items along the cross axis

align-items:flex-start; top side align-items:flex-end; bottom side align-items:center; align-items:stretch; stretch to fit in flexbox align-items:baseline; letters written

flex-wrap Property to Wrap a Row or Column

Shrink Property= flex-shrink

Opposite is flex-grow

flex-basis Property to Set the Initial Size of an Item

flex shorthand writing

flex: 1 0 10px; will set the item to flex-grow: 1;, flex-shrink: 0;, and flex-basis: 10px;.

Use the order Property to Rearrange Items The order property is used to tell CSS the order of how flex items appear in the flex container

Use the align-self Property

align-self accepts the same values as align-items and will override any value set by the align-items property.
